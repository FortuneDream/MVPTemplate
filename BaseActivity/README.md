## MVPTemplate
利用Studio的模板功能做的一个简单的MVP常用模板
### 使用步骤
* 首先在AndroidStudio的安装目录找到template目录下的activities，例如：E:\AndroidStudio\plugins\android\lib\templates\activities
* 建立一个任意文字的文件夹
* 将本项目的所有文件放入此文件夹中
* 打开AndroidStudio
